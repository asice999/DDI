# 合力数据DDI管理系统

**合力数据DDI管理系统**是一套基于 Web 的网络基础设施管理平台，集成 DNS、DHCP、IPAM 三大核心模块，为企业网络运维提供一站式可视化解决方案。

## 功能清单

1. 仪表盘总览 — 全局统计面板与多维度图表可视化
2. IP地址管理 — 区域/VLAN/子网/IP全生命周期管理与批量分配
3. 网络自动发现 — Ping/端口扫描、异步任务、结果分析与导出
4. DNS服务管理 — 正反向区域/记录CRUD、服务启停与探测监控
5. DHCP地址池 — 地址池/租约/排除规则管理与服务控制
6. 设备资产台账 — 设备注册分类、IP关联绑定与搜索
7. 多角色权限 — 管理员/操作员/只读三级权限体系
8. 审计日志追溯 — 全量操作记录，按模块/时间/用户筛选

**技术栈**: Django 4.x + Bootstrap 5 + Chart.js
