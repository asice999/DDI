# 合力数据DDI管理系统 — 软件功能介绍

> **版本**: v1.0.0  
> **更新日期**: 2026-04-18

---

## 系统简介

**合力数据DDI管理系统（Network Management DDI）** 是一套基于 Web 的网络基础设施管理平台，集成 **DNS（域名解析）、DHCP（动态地址分配）、IPAM（地址管理）** 三大核心功能模块，提供从 IP 地址规划、设备资产管理到 DNS/DHCP 服务运维的一站式解决方案。

系统采用 **Django + Bootstrap 5 + Chart.js** 技术栈，界面简洁直观，支持多角色权限控制、操作审计日志、实时告警等企业级能力。

---

## 功能概览

| 模块 | 核心能力 |
|------|---------|
| **仪表盘 Dashboard** | 全局数据总览、多维度图表可视化、实时告警 |
| **IP 地址管理 IPAM** | 区域/VLAN/子网/IP 全生命周期管理、批量分配、自动发现扫描 |
| **DNS 管理 dnsmgr** | 正向/反向区域管理、多类型记录 CRUD、服务状态监控与探测 |
| **DHCP 管理 dhcpmgr** | 地址池管理、租约跟踪、排除规则、服务启停控制 |
| **设备管理 devices** | 设备资产台账、主机名/MAC/OS 管理、IP 关联绑定 |
| **用户管理 accounts** | 多角色账号体系（管理员/操作员/只读）、密码策略 |
| **审计日志 logs** | 全量操作记录追溯、按模块/时间/用户筛选 |

## 一、仪表盘 Dashboard

登录后首页即为系统仪表盘，提供全局运行状态一目了然的总览视图：

- **统计卡片面板**：展示 IP 总数 / 已分配 / 子网数 / DNS 记录 / DHCP 池 / 活跃租约 / 设备数等核心指标
- **IP 使用率环形图**：以环形图直观呈现已分配 / 空闲 / 保留 / 冲突的占比分布
- **各子网使用情况柱状图**：堆叠柱状图对比每个子网的已分配 vs 可用数量
- **DNS 记录类型分布饼图**：统计 A / AAAA / CNAME / MX / TXT 等记录类型的占比
- **VLAN 地址分布图**：横向柱状图展示各 VLAN 下 IP 资源占用情况
- **系统告警面板**：自动检测高使用率子网（>80% 警告 / >90% 危险）和 IP 冲突风险
- **最近操作记录**：实时显示最近 15 条系统操作日志

## 二、IP 地址管理（IPAM）

### 2.1 层级化地址组织

采用 **区域 → VLAN → 子网 → IP地址** 四级层级结构，支持大规模网络的规范化管理。

- **区域管理**：创建地理或逻辑区域分组（如"北京机房"、"上海办公"），便于跨地域网络规划
- **VLAN 管理**：配置 VLAN ID、名称、网关地址及用途说明
- **子网管理**：创建子网时根据 CIDR 自动生成完整 IP 地址清单，网关自动标记为保留状态；支持查看子网详情页（IP 地址池），可按状态筛选（已分配/空闲/保留/冲突）、关键字搜索

### 2.2 IP 地址全生命周期

- **IP 分配**：单 IP 分配时填写主机名、MAC 地址、设备类型、部门等信息
- **批量分配**：指定起始/结束 IP 范围，一次性将范围内所有空闲 IP 统一置为已分配状态
- **IP 释放**：一键释放已分配 IP 回空闲池，清空关联信息
- **状态标记**：手动设置保留 / 冲突 / 禁用等特殊状态
- **全局搜索**：支持按 IP 地址 / 主机名 / MAC / 设备名复合检索，分页展示

### 2.3 网络探测与自动发现（高级功能）

内置完整的网络扫描引擎，无需依赖外部工具即可完成网络资产发现：

- **Ping 探测**：ICMP Ping 检测主机在线状态，获取延迟 / 丢包率 / TTL
- **端口扫描**：TCP 连接式端口扫描，识别开放端口及服务指纹
- **完整扫描**：Ping + DNS 反解 + 端口扫描 + ARP 查表的综合探测
- **后台异步执行**：大范围扫描任务在后台线程运行，前端 AJAX 实时轮询进度条
- **结果分析**：自动标记新发现主机（系统中未登记的在线 IP）、状态冲突（已分配但实际不在线）
- **CSV 导出**：扫描结果一键导出为 CSV 格式报告
- **快速工具栏**：IP 列表页面提供快捷 Ping 和端口扫描按钮，即时返回结果
- **自动发现规则**：配置定时调度任务，周期性对目标子网进行自动扫描
- **实时网络拓扑**：基于最近探测数据按子网分组展示在线主机全景

## 三、DNS 管理（dnsmgr）

### 3.1 DNS 服务监控

- **服务状态看板**：实时展示 DNS 服务运行状态（运行中/已停止/异常）
- **一键启停**：通过 Web 界面直接控制 DNS 服务进程的启动和停止
- **解析测试**：输入域名进行在线 DNS 解析测试，验证解析链路是否正常
- **查询日志**：记录 DNS 查询请求日志，支持清空历史

### 3.2 区域管理（Zone）

- 支持创建正向区域（如 `example.com`）和反向区域（如 `168.192.in-addr.arpa`）
- 区域详情页集中展示该区域下全部 DNS 记录
- 删除区域时级联清理其下所有记录（二次确认防误删）

### 3.3 记录管理（Record）

支持完整的 DNS 记录类型管理：

| 记录类型 | 用途 | 说明 |
|---------|------|------|
| A | 主机名 → IPv4 | 最常用的正向解析记录 |
| AAAA | 主机名 → IPv6 | IPv6 正向解析 |
| CNAME | 别名映射 | 域名别名指向 |
| MX | 邮件交换 | 邮件服务器路由 |
| TXT | 文本文本 | SPF / DKIM / 验证等用途 |
| NS | 名称服务器 | 域名权威 NS 声明 |
| PTR | 反向解析 | IP → 主机名反向映射 |

每条记录支持启用/禁用切换，禁用的记录不会写入实际配置。编辑表单针对不同记录类型智能展示对应字段（如 MX 显示优先级、PTR 显示 IP 等）。

### 3.4 DNS 探测

- **服务健康探测**：定时向配置的目标 DNS 服务器发送探测请求，监测响应时间和可用性
- **探测任务管理**：创建持久化探测计划，支持增删改查
- **可视化结果**：以表格形式展示每次探测的响应时间、状态码等详细指标

## 四、DHCP 管理（dhcpmgr）

### 4.1 地址池管理

- 创建 DHCP 地址池，定义 IP 分配范围（起始 IP ~ 结束 IP）
- 设置租约时长、默认网关、DNS 服务器、域名等 DHCP Option 参数
- 支持启用 / 禁用地址池，灵活控制哪些网段参与 DHCP 分配
- 地址池详情页展示该范围内所有排除地址和活跃租约

### 4.2 租约管理

- **租约列表**：展示所有当前有效的 DHCP 租约信息
- **手动创建**：为特定 MAC 地址预分配固定 IP（静态绑定）
- **释放租约**：强制终止某个租约，回收 IP 地址
- **过期检查**：自动检测并清理已过期但未正确释放的僵尸租约

### 4.3 排除地址

- 将某些 IP 标记为排除地址（如打印机、路由器等静态设备使用的 IP）
- 排除地址不会被 DHCP 动态分配给客户端

### 4.4 服务控制

- **服务状态面板**：实时显示 DHCP 服务运行状态
- **Web 启停**：一键启动或停止 DHCP 服务进程
- **操作日志**：记录每次服务启停操作及执行结果

## 五、设备管理（devices）

### 5.1 设备资产台账

- **设备注册**：登记主机名、设备名称、MAC 地址、操作系统、所属区域等信息
- **设备分类**：支持服务器 / PC / 笔记本 / 打印机 / 交换机 / 路由器 / 防火墙 / 摄像头 / 无线AP / 存储设备 / 其他 共 11 种设备类型
- **关联绑定**：设备可与 IP 地址建立关联关系，形成 IP ↔ 设备的双向映射（支持搜索选择）
- **详情展示**：设备详情页展示基本信息及关联的网络接口（NIC）

### 5.2 快速关联 IP

在设备编辑页面可通过下拉框（带模糊搜索）选择已有 IP 地址进行关联，或在设备详情页通过专用入口快速绑定 IP。

## 六、用户管理与权限（accounts）

### 6.1 多角色权限体系

| 角色 | 权限范围 |
|------|---------|
| **超级管理员 (admin)** | 全部功能访问权限，包括用户管理和系统配置 |
| **普通操作员** | 业务模块读写权限（IPAM / DNS / DHCP / 设备），不可管理用户 |
| **只读用户** | 仅可浏览数据和报表，无任何修改权限 |

### 6.2 用户管理

- 用户列表展示账号、姓名、手机号、角色、最后登录时间
- 新建用户时可设置初始密码
- 编辑用户时支持修改密码（留空则保持原密码不变）
- 支持 **启用 / 禁用** 账号，被禁用账号无法登录
- 密码要求：至少 8 位，必须包含字母和数字

---

## 七、审计日志（logs）

- 自动记录系统中所有关键操作的日志，包含：**操作时间、操作人、所属模块、操作类型、变更内容**
- 支持按模块（IPAM / DNS / DHCP / 设备 / 系统）和时间范围筛选
- 日志内容清晰展示操作前后的值变更（old_value → new_value）

---

## 八、通用特性

- **响应式布局**：基于 Bootstrap 5，适配桌面端和平板设备
- **左侧导航菜单**：6 大功能分组 + 二级子菜单，精确高亮当前所在位置
- **消息提示**：操作成功/失败均通过 Toast 消息即时反馈
- **二次确认删除**：所有删除操作需经过确认弹窗，防止误操作
- **分页与搜索**：列表页统一支持关键字搜索和分页浏览
- **数据校验**：前后端双重校验（如 MAC 地址格式、CIDR 合法性、密码强度等）
